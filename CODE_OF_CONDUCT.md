# Code of Conduct

## Our Pledge

We want participation in this project to be respectful, constructive, and harassment-free for everyone.

## Expected Behavior

- Be respectful and professional.
- Accept feedback in good faith.
- Focus on the work, not personal attacks.
- Help keep discussions technical and useful.

## Unacceptable Behavior

- Harassment or abusive language
- Personal attacks
- Discrimination of any kind
- Publishing private information without consent
- Trolling, intimidation, or sustained disruption

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, commits, code, and other contributions that violate this Code of Conduct.
